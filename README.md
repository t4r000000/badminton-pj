# 🏸 バドミントン ダブルス対戦組み合わせ (badminton-pj)

バドミントンのダブルス対戦組み合わせを自動生成する Web アプリ。
**完全オフライン動作 (PWA)** ・データは **localStorage** のみ、サーバー不要。

## 主な機能

- プレイヤー管理（追加 / 一括追加 / 有効・無効切替 / 削除）
- コート数の設定
- 一度組んだペア・対戦はできるだけ避ける組み合わせ生成
- 対戦履歴を画面下部に蓄積、ラウンド単位で削除可能
- 各試合で **勝敗をクリックで記録**、プレイヤーごとに勝敗集計
- プレイヤー名を **ドラッグ&ドロップ** で同ラウンド内で入れ替え（履歴・勝敗も自動再計算）
- データは `localStorage` に自動保存・復元
- **PWA 対応**: オフラインで動作、ホーム画面にインストール可能

## セットアップ

```bash
pnpm install
```

## 開発サーバー

```bash
pnpm dev
```

## 本番ビルド（オフライン利用用）

```bash
pnpm build
pnpm preview   # ビルド結果をローカル配信して確認
```

`dist/` ディレクトリ配下の静的ファイルをそのまま任意の Web サーバーにデプロイすれば、一度アクセスすれば以降はネットワーク遮断状態でも動作します。

### オフライン動作の確認手順

1. `pnpm build && pnpm preview`
2. ブラウザで `http://localhost:4173/` を開く
3. DevTools の Application → Service Workers で `activated and is running` を確認
4. 一度ページを更新してキャッシュ完了後、ネットワークをオフラインにしても動作継続

### ホーム画面に追加 (PWA インストール)

- Chrome: アドレスバー右側のインストールアイコンから
- iOS Safari: 共有メニュー → 「ホーム画面に追加」

## データ

すべてのデータは `localStorage` の `badminton-pj-state-v2` キーに保存されます。サーバーには一切送信されません。
